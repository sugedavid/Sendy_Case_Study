# sendy_case_study

A Flutter project.

## Getting Started

This project can be run on Android, iOS and Web platforms.

To run it simply create a launch.json file and copy the sample below:

```
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Sendy_Case_Study",
      "request": "launch",
      "type": "dart"
    },
    {
      "name": "Sendy_Case_Study (profile mode)",
      "request": "launch",
      "type": "dart",
      "flutterMode": "profile"
    },
    {
      "name": "Sendy_Case_Study (release mode)",
      "request": "launch",
      "type": "dart",
      "flutterMode": "release"
    }
  ]
}
```
